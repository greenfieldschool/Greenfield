# Greenfield School Website — Project Plan

## Goals
- Build a modern, highly interactive, and engaging school website that feels premium and globally competitive.
- Maintain a clean, fast, accessible experience on mobile and desktop.
- Make content easy for non-technical staff to update.

## Brand + Visual Direction
### Color System
- Primary (Deep Green): #0B3D2E
- Accent (Golden Yellow): #D4A017
- Neutrals:
  - Background: #F8FAFC
  - Surface/Card: #FFFFFF
  - Text: #0F172A
  - Borders: #E2E8F0

### Design Principles
- Clean, spacious layouts with strong typography.
- Golden Yellow used primarily for calls-to-action and highlights.
- Deep Green used for navigation, section anchors, and brand moments.
- Motion is subtle and purposeful (micro-interactions, gentle reveals).

## Target Audiences
- Prospective parents/guardians
- Prospective students
- Current families
- Alumni
- Staff applicants

## Primary User Journeys
- Prospective parent: Home → Academics → Admissions → Book a Visit / Apply
- Student: Home/Student Life → Clubs/Sports → Events
- Alumni: About → Alumni (optional) → Stories/News
- Staff: About → Careers (optional) → Apply

## Key Interactive Features
### Home Page Interactions
- Hero section with strong headline, CTA buttons, and either:
  - Lightweight video background (optimized), or
  - Image montage with animated transitions.
- Animated stats counters (e.g., student-teacher ratio, achievements).
- Testimonial slider with keyboard navigation.

### Admissions Conversion Features
- Clear admissions CTAs on every relevant page.
- “Visit / Request Info” form with step-based flow.
- Optional: program/level recommender (simple quiz).

### Events + News
- Filterable events list by category.
- “Add to calendar” links for events.
- Search for news posts.

### Gallery
- Filter chips (e.g., Sports, Arts, Science, Campus).
- Lightbox with swipe on mobile + keyboard support.

## Content Requirements (Checklist)
- Brand assets: logo (SVG preferred), favicon set, school crest (if any)
- Photography: campus, classrooms, labs, sports, arts, leadership portraits
- Copy:
  - Mission statement
  - Admissions process & requirements
  - Program descriptions
  - Contact details
  - Fees/scholarships (if public)
- Policies (as needed): privacy policy, cookies policy

## Accessibility & Performance Requirements
- WCAG 2.1 AA target
- Keyboard navigable components (menus, modals, sliders)
- Color contrast checked (especially Yellow on White)
- Performance budget:
  - Optimize images (next/image)
  - Limit heavy animations and large videos
  - Core Web Vitals-focused (LCP/CLS/INP)

## SEO Requirements
- Proper metadata per page (title, description)
- Open Graph + Twitter cards
- Structured data for organization/school
- Clean URLs and internal linking
- Sitemap + robots.txt

## Analytics & Measurement
- Track:
  - CTA clicks (Book a Visit, Apply, Request Info)
  - Form submissions
  - Top entry pages and conversions

## Delivery Roadmap
### Phase 0 — Discovery (1–3 days)
- Gather content + assets (copy, images, brand files)
- Confirm school levels offered (primary/secondary/both)
- Confirm must-have forms and admissions workflow

### Phase 1 — UX + Visual System (3–7 days)
- Finalize site map and navigation structure
- Wireframes for Home + Admissions + Academics
- Component library direction (buttons, cards, headers, forms)

### Phase 2 — MVP Build (1–3 weeks)
- Implement core pages
- Implement CMS/content workflows
- Build News & Events and Gallery
- Build forms with spam protection

### Phase 3 — QA + Launch (3–7 days)
- Cross-browser testing
- Accessibility audit and fixes
- Performance optimization
- SEO checklist verification
- Deploy to production

### Phase 4 — Enhancements (post-launch)
- Virtual tour v2 (360 embeds)
- Careers page
- Alumni stories section
- Multilingual support
